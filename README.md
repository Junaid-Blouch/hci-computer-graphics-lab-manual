## 📚 Overview

This repository contains a complete lab manual for the **HCI (Human‑Computer Interaction) and Computer Graphics** course.  
It covers two major domains:

### 🎨 Part 1 – UI/UX Design with Figma
- Figma interface (Layers, Design, Tools panels)
- Designing a login page (layout, colors, typography)
- Constraints for responsive design (NAT vs Bridged approach)
- Shape tools & object handling (selection, grouping, locking)
- Shape manipulation (fills, strokes, effects, Boolean operations)
- Color theory & typography (type scales, color/text styles)
- Low‑fidelity & mid‑fidelity wireframes
- Layout grids for desktop, tablet, mobile
- Dashboard design (12‑column grid, plugins)
- Auto Layout (buttons, cards, responsive wrapping)
- Components, variants, component sets
- Prototyping (interactions, transitions)

### 💻 Part 2 – Computer Graphics with p5.js
- Drawing primitives (points, lines, rectangles, circles, ellipses, triangles)
- Line drawing algorithms: DDA (Digital Differential Analyzer) & Bresenham (simple & full versions)
- Midpoint circle algorithm (8‑way symmetry)
- 2D transformations (translation, scaling, rotation)
- Color & visual attributes (stroke, fill, color modes, transparency)
- Bézier curves (control points, curve generation)
- Animation & motion (bouncing ball with collision detection)

All tasks include objectives, step‑by‑step procedures, code examples (p5.js), screenshots, and HCI/principles explanations.
